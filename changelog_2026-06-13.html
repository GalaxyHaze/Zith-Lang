v0.2.0 — Where It All Begins

First Changelog • June 13, 2026 • Commits: 10 • Diff: +9789 / -4272 across 112 files

Welcome to the very first Zith changelog! We're just getting started. From parser overhauls to a brand-new "did you mean?" engine — we've been busy. Zith is growing fast, and we want you along for the ride. Check back often.

✦ Parser Gets Glow-Up

  • Parser: +1700 lines of pure parse power. Scanner rework with scan-result.hpp and parse-result.hpp.
  • Tests: New parser-full.cpp (857 lines) and parser-scan.cpp (269 lines).

✦ AST Gets a Personality

  • AST: Dedicated AST nodes for enum, union, component, trait, interface.
  • AST: Builder and printer refactored — 670 lines added, 500 removed.

✦ The "Did You Mean?" Engine

  • Diagnostics: HeuristicEngine — Levenshtein-powered suggestion engine for typos.
  • Diagnostics: Error codes reorganized, emit system rewritten.

✦ Type System Goes to the Gym

  • Types: Type interning rewritten (+202 lines).
  • Types: Unification engine expanded (+275 lines). New type kinds.
  • Tests: type-dedup.cpp and unify-errors.cpp.

✦ Sema Gets Serious

  • Sema: Pipeline grew by 387 lines. Type-checks enums, unions, traits, interfaces.
  • Tests: New sema-hir.cpp (250 lines).

✦ Lexer Did a Speedrun

  • Lexer: Keyword table rebuilt (450 lines), lexer restructured (650 lines), tokens refactored.

✦ CLI Pipeline Unclogged

  • CLI: Compilation session refactored for multi-stage pipeline. New flags, updated commands.

✦ IR Trilogy (HIR → MIR → Tears)

  • ZIR: HIR, MIR, lowering bridge refactored. Cleaner types and instructions.

✦ Memory Layer Spring Cleaning

  • Memory: Arena, DynArray, Optional, Result, Span, StringInterner, SourceFile, SourceMap — all unified.

✦ Import/Export Logistics

  • Import: Manager and resolver reworked. Symbol table upgraded.
  • Tests: import-basics.cpp and import-e2e.cpp expanded (+490 lines).

✦ Infrastructure & Docs

  • Infra: CI workflows, Scoop manifest, install scripts updated.
  • Docs: Architecture plan, comparison guide, API docs, working examples added.

✦ Stdlib

  • Stdlib: Cleaned up duplicate in stdlib/unstable/json.zith.

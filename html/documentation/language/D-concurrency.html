Concurrency & Threads

Zith provides structured concurrency with compile-time safety enforcement. Thread safety is guaranteed through ownership rules and the type system.

Spawning

// Default thread
let handle = spawn worker_fn(data);

// Explicit thread type capability
let handle = spawn GreenThread fn worker_fn(data);

// Fire-and-forget
let _ = spawn background_task();

// #wontRemain — promise the thread dies before scope ends
#wontRemain let _ = spawn quick_task(shared_data);

Compile-Time Safety Enforcement

If a thread accesses shared data, the compiler requires one of:

  • await handle — before the shared data goes out of scope.
  • #wontRemain attribute on the spawn.

Missing both is a compile error. Violating the #wontRemain promise is not caught — the shared data may become a dangling reference.

ℹ Alternative: Use global: share T or Rc for runtime reference-counted sharing.

Keyword / Method Semantics
await handle Waits for the thread to finish.
await globalVar Blocks until the global variable has been initialised.
handle.send(msg) Sends a message to the running thread.
#wontRemain Attribute on spawn. Promise: thread dies before enclosing scope ends. 
let handle = spawn worker(shared_data);
await handle;

global cfg: Config;
await cfg;
@println(cfg.host);

Globals & Thread Safety

Global variables must satisfy thread-safety requirements through capabilities:

  • global: share T — requires the Share capability. Multiple threads may read and write.
  • global: unique T — requires the Lent capability (atomic operations). Runtime-checked exclusive borrow.
global score: share i32 = 0;       // requires Share trait
global counter: unique i32 = 0;    // requires Lent trait (atomic access)

NRA Limitations & Escape Hatches

NRA cannot statically validate all shared/view cycles across threads. Three approaches are available:

await — Structured Concurrency

let handle = spawn worker(shared_data);
await handle;                        // compiler verifies thread is done before scope ends

#wontRemain — Compile-Time Promise

#wontRemain let _ = spawn quick_task(shared_data);  // promise: thread dies before scope ends

Rc — Runtime Ref-Count Fallback

let shared = Rc.new(HeavyResource.init());
let _ = spawn worker(Rc.clone(shared));              // runtime reference counting

Quick Start

Step 1: Install Zith

To start your journey with Zith, use one of the following commands in your terminal:

Build from source:

git clone https://github.com/GalaxyHaze/Zith-Lang.git
cd Zith-Lang
cmake -S . -B build
cmake --build build -j

Scoop (Windows):

bucket add zithc https://github.com/GalaxyHaze/Zith-Lang.git
scoop install zithc

Homebrew (macOS/Linux):

brew tap galaxyhaze/zithc
brew install zithc

Step 2: Create a New Project

Once zithc is installed, initialize your first project:

zithc create hello-world
# Creates:
#   hello-world/
#     ZithProject.toml
#     src/main.zith
#     (and other folders that will be explained later)
Project Configuration

Step 3: Write Your First Program

Open src/main.zith. You will see the auto-generated template:

// Auto-gen 'created' "hello-world"
// Place your code here
from std/io/console

fn main() {
    @println("Hello 'hello-world'!");
}

What's going on?

  • from std/io/console: Imports the console functions (like @println and input) from the standard library.
  • //: Denotes a single-line comment.
  • @: Denotes a macro call. If you are wondering what macros are, check out the Macros specification. For now, think of it as compiler magic!
  • fn main(): The entry point where your program starts executing. Every compilable application must have exactly one main function.

Now, let's replace the content of src/main.zith with this basic calculator code:

from std/io/console

fn main() {
    @println("Hello Zith!");
    let name = "Developer";
    let age = 25;
    @println("Welcome, {name}! You are {age} years old.");

    // Simple Calculator math strings
    @println("10 + 5 = {10 + 5}");
    @println("10 - 5 = {10 - 5}");
    @println("10 * 5 = {10 * 5}");
    @println("10 / 5 = {10 / 5}");
}

Step 4: Run Your Program

Execute your project instantly, or build and run it manually:

zithc run hello-world

# Or compile and execute as separate steps:
zithc build hello-wolrd
zithc execute hello-world

Expected output:

Hello, Zith!
Welcome, Developer! You are 25 years old.
10 + 5 = 15
10 - 5 = 5
10 * 5 = 50
10 / 5 = 2

Understanding the Basics

Functions

Functions use trailing expressions (implicit returns). The last expression evaluated in a block is automatically returned without a keyword:

// Implicit return (no semicolon)
fn add(a: i32, b: i32): i32 { a + b }

// Void function (no return value)
fn greet(name: string) {
    @println("Hello, {name}!");
}

Variables and Types

Zith is statically typed with strong type inference:

// Explicit type annotations
let count: u32 = 42;
let pi: f64 = 3.14159;
let message: []char = "Hello";

// Type inference
var inferred = 100;   // i32 by default, can re-assign (var)
let text = "World";   // []char, can't re-assign (let)

Control Flow

let score: i32 = 85;

if (score >= 90) {
    @println("Excellent!");
} else (score >= 70) {
    @println("Good job!");
} else {
    @println("Keep practicing!");
}

// Ranged for loop
for (i in 0..10) {
    @println("Count: {i}");
}

// Conditional for loop
for (var i: i32 = 0, i < 10, i++) {
    @println("Count: {i}");
}

Try These Exercises

Exercise 1: Interactive Calculator

from std/io/console

fn main() {
    // Note: 'must' halts execution if string parsing fails
    let a: i32 = must input().toI32();
    let b: i32 = must input().toI32();

    @println("Sum: {a + b}");
    @println("Difference: {a - b}");
    @println("Product: {a * b}");
    @println("Quotient: {a / b}");
    @println("Remainder: {a % b}");
}

Exercise 2: Temperature Converter

fn celsiusToFahrenheit(c: f64): f64 {
    (c * 9.0 / 5.0) + 32.0
}

fn main() {
    let tempC: f64 = 25.0;
    let tempF = celsiusToFahrenheit(tempC);
    @println("{tempC}C = {tempF}F");
}

Exercise 3: Factorial

fn factorial(n: u32): u64 {
    for ([a, i]: u64 = 1; i in 1..=n) {
        a *= i
    }
}

fn main() {
    let num: u32 = 5;
    @println("{num}! = {factorial(num)}");
}

Common Commands

Command Description
zithc create <name> Creates a new project.
zithc run Runs the project configuration in ZithProject.toml.
zithc build Compiles your project.
zithc check Validates code and catches errors without outputting binaries.
zithc fmt Formats code files to style specifications.
zithc clean Removes cache and build artifacts.

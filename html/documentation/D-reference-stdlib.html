Standard Library

The Zith standard library is organised in three tiers:

  • std — Stable. Guaranteed backward compatibility within an edition.
  • utils — Experimental. May change between versions.
  • c — C bindings. FFI wrappers for common C libraries.

Importing Modules

from std.io.console;     // console I/O macros
from std.mem.arena;      // arena/scene allocation
from std.collections;    // Vec, Map, Set
from std.fs;             // file system
from std.json;           // JSON parsing/serialization

Console I/O (std.io.console)

@println("Hello, World!");
@println("Value: {}", x);
@print("no newline");
@eprint("error message");

Collections (std.collections)

Vec<T> — Dynamic Array

let vec = Vec{ 1, 2, 3 };
vec.push(4);
let first = vec[0];

Map<K, V> — Key-Value Store

let map = Map{ "key": "value" };
map.insert("name", "Zith");
let val = map.get("name");

Set<T> — Unique Elements

let set = Set{ 1, 2, 3 };
set.insert(42);

File System (std.fs)

let file = File.open("data.txt")!;
let data = file.read()!;
file.write("output")!;

Memory & Arenas (std.mem.arena)

let arena = @allocate(Arena, 1024);
let ptr = arena.alloc::<Player>(32);

Concurrency

spawn || { @println("Hello from a task!"); };

Other Modules

  • std.json — JSON parsing and serialization
  • std.time — Timing and sleep utilities
  • std.math — Math constants and functions
  • std.net — Networking primitives

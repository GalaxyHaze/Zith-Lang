Chat Guidelines

This document outlines how to use our community chat platforms effectively.

Discord Server

Our Discord server is the primary place for real-time community discussion.

Getting Started

  1. Join the server
  2. Read the #rules channel
  3. Introduce yourself in #general

Channel Overview

Channel Purpose
#general General discussion and introductions
#help Get help with Zith development
#showcase Share your projects and get feedback
#announcements Official release and update news
#off-topic Non-Zith related chat

Rules

  1. Be respectful — Treat others as you'd like to be treated
  2. Stay on topic — Keep channel-specific discussions in their respective channels
  3. No spam — Don't flood channels with repeated messages
  4. No advertising — Ask before sharing personal projects or services
  5. Use threads — Keep conversations organized with threads
  6. Search first — Check if your question has been asked before

Getting Help

When asking for help in #help: - Describe what you're trying to achieve - Show relevant code snippets - Include error messages - Mention what you've already tried

Helping Others

  • Be patient with newcomers
  • Share knowledge freely
  • Point to documentation when available
  • Don't gatekeep — everyone starts somewhere

GitHub Discussions

For longer-form discussions, feature proposals, and Q&A:

Visit Discussions →

Use GitHub Discussions for: - Detailed feature discussions - RFCs and proposals - Q&A that benefits from threading - Ideas that need community input

Etiquette

  • Don't ping unnecessarily — Use @mentions only when relevant
  • Don't quadruple-post — If your message doesn't get a response, wait
  • Don't hijack threads — Start a new conversation for new topics
  • Respect time zones — Not everyone is online when you are
  • Keep it civil — Disagree respectfully

Getting Moderator Help

If you need assistance: - Tag @moderator in the relevant channel - Open a support ticket in #support - Contact via the project maintainers

We want Zith's community to be welcoming, helpful, and productive for everyone.

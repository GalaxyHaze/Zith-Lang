VS Code Extension

ℹ Info: The official Zith VS Code extension is available now. LSP integration is a work-in-progress — contributions welcome!

Overview

The Zith extension provides rich editing support for the Zith programming language in Visual Studio Code. It is the official extension maintained by GalaxyHaze.

  • Repository: github.com/GalaxyHaze/zith-extension
  • Website: zith-lang.org

Features

Syntax Highlighting

Full colorization for keywords, operators, strings, numbers, and comments in .zith files using the "Zith Dark" color theme with a purple and red palette.

Code Snippets

Quick templates for common Zith patterns:

  • fn — Function declaration
  • struct, enum, union — Data type declarations
  • component, trait, impl — ECS and trait system
  • if, for, while, match — Control flow
  • let, letmut — Variable bindings
  • import, from — Module imports
  • macro — Macro declarations

Dark Color Theme

"Zith Dark" — a dark theme with deep purple background (#1E1A2D) and purple/red accent colors optimized for readability.

Installation

From VS Code Marketplace

Open VS Code, go to the Extensions view (Ctrl+Shift+X), search for "Zith", and click Install.

Manual Install (.vsix)

  1. Download the latest .vsix from the releases page
  2. In VS Code, run Extensions → Install from VSIX...
  3. Select the downloaded file

LSP Server Setup

⚠ Warning: LSP integration is a work-in-progress. The extension currently provides syntax highlighting and snippets only. Language server features (diagnostics, completions, go-to-definition) will be available in a future release.

To download the zith-lsp binary (when available):

node scripts/download-lsp.js

For a specific version:

node scripts/download-lsp.js v0.2.0

Installing Zith

Getting started with Zith is quick and easy. Choose your platform below:

Quick Install (Recommended)

Windows

# Using PowerShell (admin privileges may be required)
iwr -useb https://zith-lang.org/install.ps1 | iex

Or install via Scoop:

scoop bucket add zithc https://github.com/GalaxyHaze/Zith-Lang.git
scoop install zithc

Alternatively, download the installer from GitHub.

macOS

# Using Homebrew
brew tap galaxyhaze/zithc
brew install zithc

# Or using the install script
curl -fsSL https://zith-lang.org/install.sh | bash

Linux

# Using the install script
curl -fsSL https://zith-lang.org/install.sh | bash

# Or via Homebrew
brew tap galaxyhaze/zithc
brew install zithc

# Or download from releases, ARCH = arm64 or amd64
wget https://github.com/galaxyhaze/Zith/releases/latest/download/zith-linux-amd64.tar.gz
tar -xzf zith-linux-amd64.tar.gz
sudo mv zith /usr/local/bin/

Verify Installation

After installation, verify Zith is working:

zithc --version

If you see an output, then is working

Manual Installation

From Source

If you prefer to build from source:

git clone https://github.com/GalaxyHaze/Zith-Lang.git
cd Zith-Lang
cmake -S . -B build
cmake --build build -j
sudo cmake --install build

Requirements: CMake 3.20+ and a C++20 compatible compiler (GCC 11+, Clang 14+, MSVC 2022+).

Configuration

Shell Completion

Add shell completions for better CLI experience:

# Bash
  echo 'source <(zithc completion bash)' >> ~/.bashrc

  # Zsh
  echo 'source <(zithc completion zsh)' >> ~/.zshrc

  # Fish
  zithc completion fish > ~/.config/fish/completions/zithc.fish

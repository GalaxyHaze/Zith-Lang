VS Code Extension Released

2026-06-21

I've released the first version of the official Zith VS Code extension — now available on the VS Code Marketplace as "Zith".

What's Included

  • Syntax Highlighting — full TextMate grammar for .zith files
  • Code Snippets — templates for fn, struct, enum, component, trait, impl, control flow, and more
  • Zith Dark Theme — purple/red dark color scheme
  • Language Configuration — auto-indent, bracket matching, word navigation

LSP Integration

The zith-lsp language server is also under development (v0.2.0 was tagged today). Once the LSP client code is wired into the extension, you'll get diagnostics, code completion, go-to-definition, and hover information. This is a work-in-progress and open for contributions.

Getting the Extension

Search "Zith" in the VS Code Extensions view, or download the .vsix from the zith-extension repository.

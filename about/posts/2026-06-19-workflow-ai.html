Working on Workflow-AI

2026-06-19

I've been heads-down on GalaxyHaze/Workflow-AI for the past few days — a new project that's been consuming most of my free time. It's an AI-powered workflow automation tool, and honestly it's been a lot of fun to build something in a different domain for a change.

The downside is that Zith hasn't seen much love during this period. The compiler is in a healthy state — the type system is evolving, the resolver is solid, and the new type-walker infrastructure is in place — but there's still a long road ahead. I haven't pushed any significant Zith updates because I've been context-switching hard.

That said, the v0.3.1 release captures all the fixes and improvements that accumulated before the Workflow-AI detour. The critical bug fixes alone made this release worth cutting — there were some nasty UB issues hiding in the lexer and source map that needed to be addressed.

I'll be splitting my time more evenly going forward. Zith is still the main event, but Workflow-AI has been a refreshing change of pace. More soon.

— Haze

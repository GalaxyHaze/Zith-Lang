Language Reference

Complete reference for the Zith programming language.

📄️Language Guide

Overview of the Zith programming language features.

📄️Concurrency

Zith provides structured concurrency with thread safety guarantees.

📄️Contexts (DSL Namespaces)

Contexts are isolated environments that define custom syntax, keywords, and operators for domain-specific languages (DSLs).

📄️Control Flow

Master conditionals, loops, and pattern matching in Zith.

📄️ECS & Scenes

Zith has native support for data-oriented programming through Entity Component System (ECS).

📄️Errors

Handle errors gracefully with Zith's error handling mechanisms.

📄️Functions

Learn about function declaration, parameters, return types, and advanced features.

📄️Generics

Write reusable code with Zith's powerful generics system.

📄️Intrinsics

Intrinsics provide compile-time metadata and reflection accessed via the @ operator.

📄️Memory

Deep dive into Zith's memory model, allocation, and safe memory handling.

📄️Modules

Structure large projects with modules, imports, and visibility controls.

📄️Ownership Keywords

Zith uses five explicit ownership keywords that enforce memory safety at compile-time through the Node Resource Model (NRM).

📄️Packs (Tuples & Ownership)

Packs are Zith's ownership-aware tuples that track the lifecycle of multiple values.

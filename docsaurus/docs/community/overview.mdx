# Community Hub

Welcome to the Zith community! Whether you're a developer, contributor, or curious learner, there's a place for you here.

## Get Involved

### Discord

Join our Discord server for real-time discussions, help, and updates:

- **#general** — General chat and introductions
- **#help** — Get help with Zith development
- **#showcase** — Share what you're building
- **#announcements** — Release and update announcements

[Join Discord →](https://discord.gg/rPfn9TSKBS)

### GitHub Discussions

Ask questions, share ideas, and connect with other Zith developers:

[Visit Discussions →](https://github.com/GalaxyHaze/Zith-discussions/discussions)

### GitHub Issues

Found a bug? Have a feature request? Let us know:

- [Bug Reports](https://github.com/GalaxyHaze/Zith/issues/new?labels=bug)
- [Feature Requests](https://github.com/GalaxyHaze/Zith/issues/new?labels=enhancement)
- [Security Issues](https://github.com/GalaxyHaze/Zith/security)

## Contributing

Interested in contributing to Zith? Check out our [contributing guide](/docs/community/contributing) to get started.

## Code of Conduct

We are committed to a welcoming and inclusive community. Please read our [Code of Conduct](/docs/community/code-of-conduct) before participating.
